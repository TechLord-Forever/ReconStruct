ReconStruct
===========

Qt 5 tool to help reverse engineer binary file formats, such as game files.


Building
--------
To build and run:
```
$ qmake ReconStruct.pro
$ make
$ ./reconstruct
```


Unittest
--------
To compile and run unittests:

```
$ qmake ReconStruct.pro CONFIG+=test
$ make
$ ./reconstruct-tests
```
